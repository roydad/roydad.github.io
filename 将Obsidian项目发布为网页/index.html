    将Obsidian项目发布为网页

    安装Python

    参见 Python安装方法

    安装mkdocs

    使用pip安装 mkdocs 

    pip install mkdocs

    安装mkdocs主题和周边支持

    安装 mkdocs-material 主题 

    pip install mkdocs-material

    安装 jieba 中文分词工具, 便于后面支持中文关键字过滤搜索

    pip install jieba

    配置mkdocs

    将以下内容复制, 保存为 mkdocs.yml 

    # 这部分内容按实际情况修改
site_name: 摩尔百科  # 网页主页显示名称
docs_dir: Obsidian Vault # 需要发布的obsidian项目文件夹名
copyright: Copyright &copy; 2004 - 2022 MORLAB # 网页底部信息
theme:
  logo: assets/logo.png #主页logo文件的相对路径 assets文件夹处于Obsidian Vault文件夹下
  favicon: assets/favicon.png #浏览器标签页小Logo文件的相对路径,assets文件夹处于 Obsidian Vault文件夹下

# 以下部分固定不用修改
  name: material
  language: zh
  palette:
    - scheme: default
      toggle:
        icon: material/brightness-7 
        name: 切换到深色模式
    - scheme: slate
      toggle:
        icon: material/brightness-4
        name: 切换到浅色模式

markdown_extensions:
  - pymdownx.superfences:
      custom_fences:
        - name: mermaid
          class: mermaid
          format: !!python/name:pymdownx.superfences.fence_code_format
plugins:
  - search:
      separator: '[\s\u200b\-]'
extra:
  generator: false

    将 mkdocs.yml 文件, 复制到 Obsidian Vault 文件夹同级目录, 注意不是复制到 Obsidian Vault文件夹 里面

    启动服务, 愉快地在网页上浏览Obsidian笔记

    打开命令行, 定位到obsidian项目目录, 例如 mkdocs.yml 文件和Obsidian Vault 文件夹放在 D:\project 下, 就输入以下命令 

    cd D:\project

    打开命令行, 启动项目 

    mkdocs serve

    浏览器中打开 默认项目发布地址 http://127.0.0.1:8000 开始浏览

    如需要修改发布的端口,让其它用户访问, 如使用80端口发布, 输入以下命令

    mkdocs serve -a 0.0.0.0:80
